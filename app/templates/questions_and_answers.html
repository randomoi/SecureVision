Frequently Asked Questions

To create a virtual environment with Python 3.10, use the command /usr/local/bin/python3.10 -m venv venv. This command creates a new directory called venv in which the virtual environment files will be stored. To activate this environment, run the command source venv/bin/activate.

After you have activated your virtual environment, use the command pip install --use-pep517 -r requirements.txt to install dependencies from a requirements.txt.

To configure the TensorFlow Object Detection API:

  1. Go to the TensorFlow models research directory cd models/research
  2. Launch the Protocol Buffer compiler protoc object_detection/protos/*.proto --python_out=.
  3. Set the PYTHONPATH environment variable export PYTHONPATH=$PYTHONPATH:pwd:pwd/slim
  4. Lastly, run a test script object_detection/builders/model_builder_tf2_test.py

You can run all tests at once with pytest tests or you can run them individually with pystest command followed by file name, for example, pytest tests/test_video_camera.py.
