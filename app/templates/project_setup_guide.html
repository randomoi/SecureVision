Project Setup Guide

Welcome to SecureVison! Please take a momement to review setup instructions.

System Requirements
  • Python version: 3.10.9
  • Flask 2.3.0
  • TensorFlow
  • Dependencies from requirements.txt

Running the Project

Create and activate virtual environment:

  • Navigate to the project directory cd [enter path]
  • Create virtual environment /usr/local/bin/python3.10 -m venv venv
  • Activate virtual environment source venv/bin/activate

Configure the TensorFlow Object Detection API:

  • Go to the TensorFlow models research directory cd models/research
  • Launch the Protocol Buffer compiler protoc object_detection/protos/*.proto --python_out=.
  • Set the PYTHONPATH environment variable export PYTHONPATH=$PYTHONPATH:pwd:pwd/slim
  • Run a test script object_detection/builders/model_builder_tf2_test.py
  • Navigate back to the project directory by running the command twice cd ..

Setup Gmail API tokens:

  • Initialize script flask init-gmail-tokens
  • Enter access token
  • Enter refresh token
  • Enter expiration time

Lastly start the development server by running the command flask run --host=0.0.0.0 --port=5003 and visit http://127.0.0.1:5003 to see the website.

After successfully setting up the project, you can proceed with registration.
